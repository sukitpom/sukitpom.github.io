Command for git (version control)
Create new project (from current directory): git init
Check Status: git status
Add file (status=stage): git add or git add --all" or "git add *.html
Commit to repository: git commit -m "comment"
Upload to repository (git remote): git push origin or git push origin master
Download from repository to local: git pull origin master
Create new branch: git branch
Change branch: git checkout
